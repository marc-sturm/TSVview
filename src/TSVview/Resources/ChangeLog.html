Version 0.3 (2013-09-22)

General changes:
  • Update to Qt 5.1 and Qwt 6.1.
  • Settings are now stored in appliction directory INI file (improves portability)
New features:
  • Plot now can show both the filtered and unfiltered data.
  • Added file watcher to reload data file upon modification outside TSVview.
  • Added functionality to transpose the data matrix.
  • Added column merging functionality.
  • Filters sets can be stored/loaded now.
  • Added string to numeric column conversion.
  • Added string value list to filter dialog.
  • Added inverse regexp string column filter.
  • Added copy-to-clipboard button/hotkey to plots.
Imrovements:
  • Plot headers now contain the file name.
  • A resonable filename is proposed when storing plots and in 'save as' dialog.
  • Histogram plot default bin number set to 40 (was 20).

Version 0.2 (2012-03-03)

New features:
  • Added drag-and-drop support to open single files.
  • Added Windows context menu support.
  • Added sorting in descending order.
  • Added regexp string column filter.
  • Cell values can be edited via double click/F2 key now.
Imrovements:
  • Improved file handling.
  • Plots and statistics are now based on filtered data.
  • Filters can be stored to ZXV format.
  • Curves can be hidden/show by clicking the corresponding legend item in the data plot.
